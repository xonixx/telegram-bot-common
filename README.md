# telegram-bot-common
Some common code for multiple telegram bots

## Install
 
**Note: only Java 11+ is supported**

```xml 
<project>
    <repositories>
        <repository>
            <id>jitpack.io</id>
            <url>https://jitpack.io</url>
        </repository>
    </repositories>
    <dependencies>
        <dependency>
            <groupId>com.github.xonixx</groupId>
            <artifactId>telegram-bot-common</artifactId>
            <version>0.0.4</version>
        </dependency>
    </dependencies>
</project>
```
